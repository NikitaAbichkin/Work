# эндпоинты

База: `http://localhost:8080`

Все ответы обёрнуты в `ApiResponse`. Большинство эндпоинтов требуют `Authorization: Bearer <access_token>`.

## Auth

`POST /api/v1/auth/register`  
Регистрация. Тело: `username`, `password`, `email`. Возвращает сообщение о подтверждении.

`POST /api/v1/auth/login`  
Логин. Тело: `username`, `password`. Возвращает `TokenResponse`.

`POST /api/v1/auth/confirm`  
Подтверждение кода. Тело: `username`, `code`.

`POST /api/v1/auth/resend`  
Повторная отправка кода. Тело: `email`.

`POST /api/v1/auth/refresh`  
Обновление токена. Тело: `refreshToken`.

`POST /api/v1/auth/logout`  
Логаут. Тело: `refreshToken`.

`POST /api/v1/auth/sendPasswordResetCode`  
Отправка кода сброса. Параметр: `email`.

`POST /api/v1/auth/verifyCode`  
Проверка кода + смена пароля. Тело: `email`, `code`, `newPassword`.

`POST /api/v1/auth/deleteUser`  
Удаление пользователя. Тело: `token`.

`POST /api/v1/auth/logoutFromEverySession`  
Выйти из всех сессий. Параметр: `token`.

`POST /api/v1/auth/seeAllSessions`  
Список сессий. Параметр: `token`.

## Goals

`POST /api/v1/goals`  
Создать цель. Тело: `title`, `description`, `priority`, `start_date`, `deadline`, `daily_time_minutes`, `stages`.

`GET /api/v1/goals`  
Список целей с пагинацией. Query: `page`, `size`.

`GET /api/v1/goals/{id}`  
Одна цель со всеми этапами.

`PATCH /api/v1/goals/{id}`  
Частичное обновление цели. Тело: любые изменяемые поля.

`DELETE /api/v1/goals/{id}`  
Удалить цель.

`GET /api/v1/goals/goals`  
Поиск по параметрам. Тело через `@ModelAttribute`: `ParametersForSearching`.

## Tasks (Stages)

`POST /api/v1/goals/{goalId}/tasks`  
Создать задачу. Тело: `title`, `description`, `priority`, `estimatedMinutes`, `deadline`, `startsAt`, `sortOrder`.

`PUT /api/v1/goals/{goalId}/tasks/{stageId}`  
Обновить задачу. Тело: любые изменяемые поля.

`DELETE /api/v1/goals/{goalId}/tasks/{stageId}`  
Удалить задачу.

`GET /api/v1/goals/{goalId}/stages`  
Список задач цели. Query через `@ModelAttribute`: `Allstages`.

`GET /api/v1/goals/{goalId}/tasks/{stageId}`  
Одна задача по ID.

## Results

`GET /api/v1/goals/{goalId}/results`  
Список результатов цели.

`POST /api/v1/goals/{goalId}/results`  
Создать результат. Тело: `ResultCreateRequest`.

`PATCH /api/v1/goals/{goalId}/results/{resultId}`  
Обновить результат. Тело: `ResultCreateRequest`.

`DELETE /api/v1/goals/{goalId}/results/{resultId}`  
Удалить результат.

## AI

`POST /api/v1/goals/ai-decompose`
Генерация плана цели с нуля через AI. Тело: `prompt`. Возвращает `AIPlanResponce` (title, description, priority, stages и т.д.). Не сохраняет в базу — фронт показывает результат, юзер подтверждает, затем шлёт в `POST /api/v1/goals`.

`POST /api/v1/goals/ai-help`
Изменение существующей цели через AI. Тело: `prompt`, `goalId`. AI получает текущую цель из базы и возвращает изменённый план.
